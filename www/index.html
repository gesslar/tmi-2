Welcome to TMI

The Mud Institute is a body dedicated to the advancement and support of muds and their users. The word "mud" originates from the acronym MUD, meaning Multi User Dungeon. It has come to mean any multiuser virtual reality, whether used as a game or not. eg Abermud, BSX, DGD, Dikumud, LPmud, MOO, MUCK, MUG, MUSE, MUSH, etc.

TMI's mud

 The Mud Institute runs a mud of its own: TMI-2. Anyone can log in to this, and all users may see and write code on it. From this page you can interact with the mud in a number of ways:
  • Log in
  • See who is logged in
  • Finger someone who is logged in

About these pages

 These web pages are being brought to you by a web server written in LPC, running from inside the mud using MudOS's TCP/IP socket efuns. We have another home page that has information on the people behind TMI.
